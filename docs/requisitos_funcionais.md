# Requisitos Funcionais

RF01 – O sistema deve permitir o cadastro de funcionários com nome, CPF, cargo, setor, salário base e data de admissão.

RF02 – O sistema deve permitir a autenticação de usuários com e-mail e senha.

RF03 – O sistema deve permitir o registro de ponto (entrada, saída, intervalo e retorno).

RF04 – O sistema deve exibir o horário de Brasília no momento do registro de ponto.

RF05 – O sistema deve registrar a localização no momento da batida de ponto.

RF06 – O sistema deve calcular automaticamente as horas trabalhadas.

RF07 – O sistema deve permitir a visualização dos registros por mês e ano.

RF08 – O sistema deve gerar relatórios de ponto em PDF.

RF09 – O sistema deve permitir o envio de justificativas.

RF10 – O sistema deve permitir informar se a justificativa possui compensação de horas.

RF11 – O sistema deve permitir o gerenciamento de justificativas pelo administrador.

RF12 – O sistema deve permitir o cadastro, edição e remoção de funcionários pelo administrador.

RF13 – O sistema deve calcular a folha de pagamento com base nas horas trabalhadas.

RF14 – O sistema deve aplicar descontos como INSS, FGTS e vale transporte.

RF15 – O sistema deve permitir a opção de adesão ou desistência do vale transporte.

RF16 – O sistema deve gerar contracheques com salário bruto, descontos e salário líquido.

RF17 – O sistema deve permitir a geração de contracheque em PDF.

RF18 – O sistema deve controlar o banco de horas.

RF19 – O sistema deve calcular adicional noturno.

RF20 – O sistema deve controlar férias dos funcionários.

RF21 – O sistema deve calcular o 13º salário.

RF22 – O sistema deve permitir a visualização dos dados do funcionário.

RF23 – O sistema deve permitir edição de algumas informações do perfil.

RF24 – O sistema deve enviar relatórios automaticamente ao administrador.
